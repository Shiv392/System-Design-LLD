# LLD Playground 🚀
A collection of **Low-Level Design (LLD)** concepts, **SOLID principles**, **Design Patterns**, and real-world **Object-Oriented Design problems** implemented in Java.

This repository serves as my personal learning journey toward becoming a better software engineer and preparing for backend and product-based company interviews.

---

## 📚 Repository Structure

```text
java-lld-playground/
│
├── design-patterns/
├── solid-principles/
├── lld-problems/
└── notes/
```

### 🎯 Design Patterns

Contains implementations of commonly used design patterns along with explanations and use cases.

Examples:

* Singleton Pattern
* Factory Pattern
* Strategy Pattern
* Observer Pattern
* Builder Pattern
* Decorator Pattern
* Adapter Pattern
* Command Pattern

---

### 🎯 SOLID Principles

Practical examples demonstrating the five SOLID principles.

* Single Responsibility Principle (SRP)
* Open/Closed Principle (OCP)
* Liskov Substitution Principle (LSP)
* Interface Segregation Principle (ISP)
* Dependency Inversion Principle (DIP)

---

### 🎯 LLD Problems

Real-world system design problems solved using object-oriented design principles.

Examples:

* Tic Tac Toe
* Parking Lot System
* Library Management System
* ATM Machine
* Snake and Ladder
* Splitwise
* BookMyShow
* Elevator System
* Vending Machine

Each problem typically includes:

* Requirements Analysis
* Class Design
* UML/Class Diagrams (where applicable)
* Java Implementation
* Design Decisions

Personal notes and interview preparation material covering:

* OOP Concepts
